Typical workflow

Step 01

Add materials

Start by adding your raw materials under Inventory — name, category, stock level, unit, and reorder threshold.

Step 02

Create recipes

Define a recipe for each product. Specify which materials it uses and the quantity required per batch.

Step 03

Run batches

Create a batch from a recipe. The system tracks each batch's status — ready, in progress, or shipped.

Step 04

Ship & export

Mark batches as shipped when dispatched. Export any data to Excel for records or reporting.

Tips

Low stock alerts — Set a reorder level on each material. The Low Stock page shows everything below that threshold so you can reorder before running out.
Manage pages — Every section has a Manage page where you can edit or delete existing records. Changes to materials or recipes don't retroactively affect already-created batches.
Exports — Excel exports are a live snapshot. Run them at any point for up-to-date records. Useful for end-of-month reporting or handing off to a supplier.
